## About SpaceSniffer
SpaceSniffer is a desktop application designed to analyze and visualize disk space usage on Windows systems. It scans storage devices and presents the results using an interactive treemap layout that shows how space is distributed across folders and files.

The software is suitable for system administrators, IT professionals, and everyday users who need a clear overview of storage consumption. SpaceSniffer helps identify large files and directories quickly without requiring complex configuration.

SpaceSniffer is known for its predictable scanning behavior and clear visual output. The treemap representation allows users to explore storage usage intuitively by navigating through blocks that represent folder and file sizes.

The application focuses on clarity and transparency, offering real-time scanning feedback and interactive exploration tools that support informed storage management decisions.
